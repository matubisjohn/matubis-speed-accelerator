Given a tight deadline for this assessment, I intentionally combined my workflow with AI to move as fast and efficiently as possible. I started by having Gemini analyze the brief and appendices to extract core requirements, map out a clear action plan, and explore initial strategic ideas for the LDV brand.

To find the best technical foundation for the webpage, I researched and benchmarked ten different AI web development platforms. Because I was working on free tier plans with strict daily credit limits, I could not afford to waste iterations; every single prompt had to count. To solve this, I used ChatGPT to analyze the project brief and engineer a highly detailed master prompt that gave the web builders explicit structural instructions. While many of the platforms generated generic, bland designs, Lovable.dev produced a highly professional layout. It was not perfect right away, so I used the best visual elements and ideas from the other nine tools as inspiration, prompting Lovable through multiple rounds of revisions until the desktop and responsive layouts looked exactly right.

Once I exhausted my platform credits on the live build, I shifted my focus to the Figma deliverable. I initially tried importing the generated code directly back into Figma, but the automated translation was completely unusable because the layers were a mess and the typography was broken. Instead of trying to patch up a broken file, I chose to rebuild the entire interface from scratch. I used that failed export strictly to harvest the AI generated photos and graphics, then manually reconstructed the layouts, built clean components, organized the layer hierarchies.

With the high fidelity Figma file and prototype completed, I took the production code provided by my work with Lovable and pushed it directly to GitHub for secure hosting and version control. This full cycle process forced me to act as both a designer and a creative product manager, balancing strict resource limits and unexpected technical tool failures to deliver a polished, responsive digital showcase on a highly compressed timeline.
